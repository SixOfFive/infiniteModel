# InfiniteModel dependencies.
#
# Install only what a given machine needs:
#
#   Controller (server.py):   pip install fastapi uvicorn
#   Worker, Milestone 1:      pip install psutil
#   Worker, Milestone 2+:     pip install psutil torch transformers safetensors huggingface_hub
#
# torch is large and its wheels lag new Python releases — install it on the
# workers only when model execution lands (M2). M1 (registry/dashboard) needs
# just the controller + psutil stack below.

# --- Controller (server.py) ---
fastapi
uvicorn

# --- Worker (client.py) — Milestone 1 ---
psutil

# --- Worker (client.py) — Milestone 2+ (model loading & execution) ---
# torch
# transformers
# safetensors
# huggingface_hub
# einops         # required by some models' trust_remote_code (e.g. nomic-embed-text encoder)
# nvidia-ml-py   # optional, only for VRAM reporting on CUDA nodes
#
# AMD GPU (ROCm) worker: do NOT use a CPU/CUDA torch — install a ROCm build matched to
# your GPU arch. For Strix Halo (gfx1151) the generic download.pytorch.org wheels crash
# in the bundled HSA runtime; use AMD's arch-specific TheRock wheels instead. The helper
#   ./install-rocm.sh [arch]      # default gfx1151
# does the whole venv (torch + matched ROCm runtime + these deps). See docs/ROCM.md.
